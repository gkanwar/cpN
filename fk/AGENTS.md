Focus all work in the `fk` subdirectory of this repository. The aim is to
implement a sampler that takes input CP(N) lattice configurations as data
and produces samples of associated flow fields according to the Feynman-Kac
procedure described in 2603.00252. Read the included pdf for context, if needed.

All code contributions should be minimal changes. Do not explicitly set `dtype`
or `device` when using Pytorch. There is no formal style guide, match the
surrounding code when possible. Avoid excess comments or spaces.
